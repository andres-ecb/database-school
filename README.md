Este repositorio consiste en el diseño y explotación de una base de datos llamada *school*. Incluye la creación completa del esquema (tablas, relaciones y datos de ejemplo) y una colección de 28 consultas SQL que abordan distintos niveles de complejidad, desde consultas básicas hasta subconsultas, funciones agregadas y lógica condicional.

# Tecnologías Utilizadas
- SQL (MySQL / MariaDB compatible)
- Sistema de gestión de bases de datos relacional (RDBMS)

# Objetivo
Este repositorio está enfocado en:
- Practicar la creación de bases de datos relacionales
- Aplicar consultas SQL reales

En cuanto a las consultas, en específico, se busca demostrar habilidad en:
- Consultas básicas (SELECT, WHERE, ORDER BY)
- Funciones de agregación (COUNT, AVG, MIN, etc.)
- Subconsultas
- JOIN entre múltiples tablas
- Manipulación de cadenas
- Condiciones lógicas (CASE, BETWEEN, IN, etc.)
